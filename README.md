# Task Manager (Python CLI)

A simple command-line task manager built using Python.

This project was created as a beginner practice exercise to work with loops, lists, conditionals, and user input in Python.

## Features
- Add tasks to a list
- View all tasks
- Remove a task by name
- Quit the program safely

## How It Works
1. The program displays a menu of options.
2. The user chooses an action:
   - Add a task
   - View all tasks
   - Remove a task
   - Quit the program
3. The program continues running until the user chooses to quit.

## Skills Demonstrated
- Python `while` loops
- Conditional statements
- List operations
- User input handling
- Basic command-line interaction

## How to Run
Make sure Python is installed, then run:

```bash
python cli_task_manager_python.py
